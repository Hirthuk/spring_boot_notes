Project Setup

We Can use Spring Initializer or Eclipse Spring Boot Wizard from Marketplace.

We have used Eclipse Wizard for Project Setup and Dependency Downloads

Help → Eclipse Marketplace → Spring Tools → Download = To Download the Spring Wizard in eclipse

Project creation
  • File → New → other → Search Spring → Spring Starter Project
  • Mention Project Name, Required Dependencies, Maven Build, Package name, Artifact name and click finish to complete project setup
